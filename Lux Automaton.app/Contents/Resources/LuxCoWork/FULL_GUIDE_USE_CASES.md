# Lux CoWork - Full Guide and Use Cases

## What this app is for

Lux CoWork is your fast conversational build layer:

- quick ideation,
- coding and workflow support,
- local MCP tool calling,
- lightweight collaboration loop.

## Recommended workflow

1. Launch CoWork from Lux Hub.
2. Keep provider on Opencode for local/free-first usage.
3. Use it for rapid drafts, planning, and implementation tasks.
4. Hand off major execution flows to Manus or Hermes as needed.

## Use Case 1 - Rapid content ideation

- Generate 20 hook variants in one pass.
- Rank by curiosity gap and relevance.
- Send top candidates to Lux Tube Shorts Lab.

## Use Case 2 - Production QA

- Review scripts for clarity and claims.
- Generate title and thumbnail angle options.
- Validate consistency with channel voice.

## Use Case 3 - Toolchain glue

- Call local tools and prepare prompts for Hermes cron jobs.
- Build reusable snippets for repeat publishing workflows.
