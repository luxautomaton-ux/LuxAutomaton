# Lux CLI Docker

Self-hosted AI coding assistant UI.

## Supported Agents

- `claude-code` — Anthropic Claude Code
- `codex` — OpenAI Codex
- `gemini` — Google Gemini CLI

## Usage

Requires the `sbx` CLI from Docker. See the main project at [github.com/lux-automaton/LuxCLI](https://github.com/lux-automaton/LuxCLI)
