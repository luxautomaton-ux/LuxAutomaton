# Lux Manus - Full Guide and Use Cases

## What this app is for

Lux Manus is your execution workspace:

- run multi-step agent tasks,
- manage files and attachments,
- inspect logs and outputs,
- install and use skills.

## Recommended workflow

1. Start Manus from Lux Hub.
2. Create a task with clear objective + constraints.
3. Attach source docs, PDFs, screenshots, or datasets.
4. Run task and monitor logs for quality control.
5. Export useful outputs and convert repeat steps to skills.

## Use Case 1 - Script polishing

- Input raw script from Lux Tube.
- Ask Manus to tighten hook, pacing, and CTA.
- Export final teleprompter-ready script.

## Use Case 2 - Research packet generation

- Upload notes, links, and transcripts.
- Ask for condensed evidence packet and action list.
- Save final packet for weekly planning.

## Use Case 3 - SOP building

- Run same workflow multiple times.
- Capture best prompts and convert to reusable skills.
