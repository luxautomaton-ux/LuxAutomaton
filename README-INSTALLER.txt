=====================================
     LUX AUTOMATON - INSTALLATION
=====================================

📦 INSTALLATION:
---------------
1. Open "Lux-Automaton-Installer.dmg"
2. Drag "Lux Automaton.app" to your Applications folder
3. Launch from Applications folder

🚀 FIRST RUN:
-------------
- The app will automatically:
  • Start all AI services
  • Open your dashboard at localhost:1337
  • Initialize Ollama (if installed)

📋 REQUIREMENTS:
---------------
• macOS 10.15 or later
• Python 3.9+
• Ollama (optional, for local AI)

🔧 FEATURES:
-----------
✅ Local AI with Ollama
✅ Video generation (Hyperframes)
✅ Viral content analysis
✅ AI agent workflows
✅ Modern web UI

🌐 ACCESS POINTS:
----------------
• Main Dashboard: http://localhost:1337
• System Scanner: http://localhost:1337/system-scanner
• Hermes Setup: http://localhost:1337/hermes-setup
• Lux Tube (YouTube): http://localhost:5173

❓ TROUBLESHOOTING:
------------------
If the app doesn't start:
1. Check if Python 3 is installed: python3 --version
2. Install Ollama: https://ollama.com
3. Check logs in Console.app

📧 SUPPORT:
----------
For issues or questions, check:
