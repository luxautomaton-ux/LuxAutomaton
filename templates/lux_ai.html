⚙️ How to Connect from Another Device

1

Find Your Local IP Address

Run ifconfig (macOS) or ip addr (Linux) to get your local IP (e.g., 192.168.1.100)

2

Ensure Hub is Running

Hub should be running on port 1337. Check: lsof -i :1337

3

Connect from Another Device

In your client tools, point to your IP:

# For Ollama-compatible tools OLLAMA_HOST=http://192.168.1.100:11434

# For OpenAI-compatible tools (like Claude Code) OPENAI_BASE_URL=http://192.168.1.100:1337/v1
4

Access Hermes Remotely

Open in browser: http://192.168.1.100:8787

5

For Outside Your Network (Port Forwarding)

Router settings → Port Forward → 1337 → Your machine's local IP

☁️ VPS Setup — Run Models on Cloud

Host llama.cpp on a VPS and connect to it from your local machine.

VPS Server Setup (Ubuntu 22.04)

# 1. Install llama.cpp server curl -fsSL https://ollama.com/install.sh | sh

# 2. Pull your model ollama pull qwen2.5:32b

# 3. Start server on all interfaces ollama serve --host 0.0.0.0

Connect from Local Machine

# Point your local tools to VPS OLLAMA_HOST=http://YOUR_VPS_IP:11434

# Or use Hub's proxy to route through VPS export OPENAI_BASE_URL="http://localhost:1337/v1"
# Then in your .env, configure the upstream

Security — Use SSH Tunnel (Recommended)

# Create secure tunnel from local to VPS ssh -N -L 11434:localhost:11434 user@your-vps-ip

# Then use localhost:11434 as if VPS was local OLLAMA_HOST=http://localhost:11434
